## 从 Accept 开始

`http.ListenAndServe` 背后是一个朴素的循环：`Accept` 一个连接，起一个 goroutine。

```go
for {
    rw, err := l.Accept()
    if err != nil {
        return err
    }
    c := srv.newConn(rw)
    go c.serve(connCtx)
}
```
