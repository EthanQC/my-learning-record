#### `sudo apt-get update` 和 `sudo apt update` 的区别
`apt-get` 是 Debian 和 Ubuntu 系统中传统的包管理工具，已经存在多年，比较老

`apt` 是较新的工具，可以看作是 `apt-get` 的升级版，输出信息会更直观易读，下载速度也会更快，在现代 Linux 系统中是首选命令
